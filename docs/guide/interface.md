# 界面介绍

## 主界面布局

NarrateFlow 采用 4 步流程式设计，从左到右依次完成。

### 步骤导航栏

顶部显示当前步骤（上传 → 场景理解 → 选段 → 导出），已完成步骤高亮。

### Step 1 · 上传视频

拖拽或点击上传，支持文件夹批量导入。自动扫描 mp4/mov/avi/webm 格式。

### Step 2 · 场景理解

视频逐帧分析，提取第一人称视角片段。可预览每个片段并调整。

### Step 3 · 情感选段

情感峰值加权排序，悬疑铺垫 → 剧情高潮 → 情感共鸣。支持手动增删调整顺序。

### Step 4 · 解说导出

选择情感风格，配置配音参数，一键生成 MP4 或剪映草稿。

## 底部状态栏

- 当前处理状态
- API Key 配置快捷入口
- 版本信息
